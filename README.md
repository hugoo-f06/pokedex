# Pokedex (Java)

Pokedex project built in Java.  
Goal: start with Gen 1 (offline JSON), then expand to more generations and later add an online mode (PokeAPI) + UI.

## Features (WIP)
- [ ] List Pokemon
- [ ] Search by name/id
- [ ] Filter by type
- [ ] Multi-generation support

## Tech
- Java 17
- Maven
